# Low-Cost-Digital-Oscilloscope-Built-from-Scratch-
Designed and built a low-cost digital oscilloscope capable of displaying clean waveforms up to 6 MHz. Implemented high-speed signal acquisition, ADC interfacing, real-time visualization, and filtering. Optimized hardware and firmware to achieve stable, accurate, and affordable signal measurement.
